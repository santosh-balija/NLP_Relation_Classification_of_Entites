To run the source code of the project, make use of Jupyter Notebook.

The files to be uploaded into the Jupyter console before running the program are:

	1) train.txt
	2) test.txt
	3) levinClass.txt
The levinClass.txt is the text file created from the offical website where we can find the levin Class classification of the verbs.
The levinClass.txt file has been attached in the project submission folder.
